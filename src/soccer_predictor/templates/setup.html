How to Configure API Keys

Important Note

API keys are stored in your environment variables and are never saved in the database. They will persist as long as your server is running.

Step-by-step Guide:
  1. Sign up for an account at football-data.org (free tier available)
  2. Obtain your API key from your account dashboard
  3. Enter your API key in the form above
  4. Click "Save Configuration"
  5. Wait a few moments for data to sync from the API
  6. Refresh the fixtures page to see live data
